Eggspress is a lightweight blogging platform built on Next.js and React. It's designed to help anyone launch a new website with little to no coding experience. There's no command line needed. Best of all, it's lightning fast and completely free![^1]

![](painting.jpeg)

## Why Eggspress?
Eggspress is a great place to start a new site. It's very lightweight and nimble, providing your readers with a snappy experience. Pages load very quickly even on slower connections. Because it uses Github and Vercel, it's also free thanks to these services' generous free tier.

Out of the box, Eggspress provides core features that cover most authors' needs. If you're a developer or have access to one, you can easily extend Eggspress' open-source code to build new features and connect to other services.

Your content belongs to you. That's why we use Github-flavored Markdown, a widely used standard, to style your text. If you ever feel that you've outgrown Eggspress, you can move to another Markdown-based platform. Just move your files!

Of course, if you do decide to use Eggspress, we'd love for you to stay. If you have feedback or suggestions, [create an issue](https://github.com/dentonzh/Eggspress/issues/new) with your comments or email your thoughts to eggspressblog [at] gmail.

## When not to choose Eggspress
Eggspress is a new platform. As a result, many features that you may expect from more established platforms may be absent. For this reason alone, you might wish to consider alternatives.

Because it's new, there are presently no themes available to help you change your layout. To make up for this, we've created a system that allows you to set color schemes easily. You can also define your own custom color scheme on over 100+ different elements.

Lastly, Eggspress is a CMS-less platform, which might not appeal to everyone. There is no sleek interface for managing your site. Every change you make to your site's appearance and every content you create must be written as text in your content files.

One side effect of this is that it's difficult to maintain an Eggspress site using a phone or tablet alone. We strongly recommend using a computer running Mac, Windows, or Linux to publish to Eggspress.

The benefit, however is that you can define all of your preferences for a piece of content in one file. If you're familiar with using keyboard-only workflows (in Excel, for example), you might find this way of doing things preferable.
## Features
- Free to deploy and host
- No-code set up and publishing
- Markdown-based
- SEO ready
- Dark/light mode
- Custom color schemes
- Supports videos (mp4 and webm)
- Custom categories, pages, and sidebars
- Multiple authors
- Excellent [PageSpeed Insights](https://pagespeed.web.dev/) benchmarks
- One-click updates
- Google Analytics
- Built-in support for affiliate links and query parameters
- Compatible with MDX and custom components
## Prerequisites
To use Eggspress, you'll need a:
- Github account ([sign up](https://github.com/join))
- Vercel account ([sign up](https://vercel.com/signup) using "Continue with Github")
- Markdown editor ([Obsidian](https://obsidian.md/) is recommended)
## Overview
If you are new to Github or are unfamiliar with starting a website, it will take you about one to two hours to complete this guide.

You should be comfortable working with computers. At a minimum, you should know know how to download files to your computer and move them into different folders. You should also know how to drag and drop files.

## What this guide covers
1. Forking ("installing") Eggspress
2. Launching your site to the web
3. Setting up your content folders on your computer
4. Uploading your content to Github
6. Creating and editing posts, custom pages, and author profiles
7. Configuring your settings and choosing a color scheme
